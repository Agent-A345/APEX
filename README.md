## 📊 APEX — Analytics Dashboard

A modern interactive analytics dashboard web app built using HTML, CSS, and JavaScript, featuring dynamic charts, KPI tracking, filtering systems, trend analysis, drill-down insights, dark mode, and export functionality for visual business intelligence.

## 🚀 Features
* **Interactive Analytics Dashboard** – Real-time dashboard with charts, KPIs, and insights.
* **Multiple Dataset Support** – Switch between sales, expenses, and productivity datasets.
* **Dynamic KPI Cards** – Displays revenue, transactions, averages, and top categories.
* **Trend Analysis Engine** – Detects growth trends and performance changes.
* **Interactive Line Chart** – Visualizes trends over time.
* **Category Comparison Charts** – Compare categories using bar and pie charts.
* **Compare Mode** – Compare current vs previous periods visually.
* **Advanced Filtering System** – Filter by month, category, and type.
* **Drill-Down Analytics** – Click charts/tables to inspect detailed records.
* **Insights Ticker** – Generates dynamic analytical insights automatically.
* **Data Table System** – Displays categorized analytics breakdowns.
* **Add Custom Data** – Insert new records dynamically into datasets.

## 🎮 How to Use
**Switch Dataset**
* Choose sales, expenses, or productivity dataset

**Change Time Period**
* View analytics daily, weekly, or monthly

**Apply Filters**
* Filter data by month, category, and type

**Enable Compare Mode**
* Compare current vs previous period trends

**Analyze Charts**
* View line, bar, and pie chart insights

**Switch Theme**
* Toggle between dark and light mode

**Export Dashboard**
* Download dashboard snapshot as image
